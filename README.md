This is an app that has benefits like : 

- 2-Factor Auth with OTP
- Activation account with OTP
- Reset password with OTP
- React library for its frontend

Before test this project you need to do some :

- first fill up some settings in `settings.py` like "SECRET_KEY".
- create `migrations` folder in user app
- add `node_modules` into frontend folder for React library
